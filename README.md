# :sparkles: 提交规范

- 🎉 init: 项目初始化
- ✨ feat：新增功能（feature）
- 🐞 fix：修复bug
- 📃 docs：文档修改
- 🌈 style：代码样式调整，不影响代码逻辑
- ✅ test: 测试相关的改动
- 🔨 refactor: 代码重构
- 🔧 chore: 构建过程或辅助工具的变动

# :dart: 兼容环境

- 现代浏览器（Chrome >= 64, Edge >= 79, Firefox >= 78, Safari >= 12）
